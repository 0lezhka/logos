Привіт! Використовуйте цей репозиторій для закидання домашнього завдання.

Порядок закидання ДЗ:
1) Створіть бренчу з вашим прізвищем та номером заняття (наприклад: oleh_kopylchak_1)
2) Увесь код повинен зберігатись у папці з вашим іменем та прізвищем. 
   Завдання на кожне заняття також повинне бути у відповідній папці.
   Наприклад:
	oleh_kopylchak_1/home_work_1/..
	oleh_kopylchak_1/home_work_2/..	
	...
3) Відкрийте pull request в main бренчу
4) Далі іде рев'ю коду. Якщо проблем немає, то викладач мерджає код в main, в іншому випадку 
   залишає коментар на пул реквесті з зауваженнями.
